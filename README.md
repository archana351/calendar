# 📅 Interactive Calendar Component

A responsive and visually polished **calendar UI component** built using React.js.
This project was developed as part of a **Frontend Engineering Task** to demonstrate strong skills in UI design, state management, and user experience.

## ✨ Features

### Core Functionality

* 📆 Dynamic calendar with **month navigation**
* 🖱️ **Date range selection** (start → end → reset)
* 🎨 Highlighting for selected range
* 📝 **Notes system** for selected date ranges
* 💾 Data persistence using **localStorage**
* 📱 Fully responsive layout

---

### Enhancements

* 🌙 Dark mode toggle
* 🎯 Current day highlight
* 🧠 Multiple notes per date range
* 🧭 Weekday headers (Sun–Sat)
* 🎞 Smooth animations and hover effects
* 🖼️ Hero image-based layout (wall calendar style)

---

## 🧠 Implementation Highlights

* Component-based architecture using React
* State lifting for shared data between components
* Efficient date handling for dynamic month rendering
* LocalStorage integration with controlled side effects
* Clean separation of UI and logic

## 📌 Key Decisions

* Date range resets on third click for better UX
* Notes are mapped using a range-based key system
* Layout adapts between desktop (split view) and mobile (stacked view)
* No backend used — data handled via client-side storage
